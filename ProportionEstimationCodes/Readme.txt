This folder contains modular functions that implement the metheods of proportion estimation for multiple testing composite null hypothesis in 
1. the publication "Uniformly consistent proportion estimation for composite hypotheses via integral equations: ``the case of Gamma random variables"" and,
2. its follow-up publication "Uniformly consistent proportion estimation for composite hypotheses via integral equations: ``the case of location-shift families''"

There are three files in this folder:
a. EstPi_ModFuncs.r
b. EstPi_ModFuncsB.r 
c. EstPi_ModFuncsCA.r
